# Simple Manga Viewer

簡潔的跨平台漫畫／插畫閱讀器，以 Electron 打造，支援本機資料夾瀏覽與多種顯示模式。

> ⚠️ 目前僅在 Windows 環境測試，macOS 尚未驗證。

## 功能特色

- **開啟目錄**：選擇本機資料夾，自動掃描並依檔名排序圖片
- **開啟壓縮檔**：直接讀取 `.zip` / `.cbz` 壓縮檔，無需手動解壓
- **檔案關聯**：在檔案總管中雙擊圖片即可直接啟動應用程式閱讀
- **側邊欄瀏覽**：顯示子資料夾與檔案列表，可快速切換章節或單檔
- **閱讀進度記錄**：自動記憶各目錄的閱讀位置，重新開啟時從上次位置繼續
- **顯示模式**
  - **適合寬度 / 適合高度**：自動縮放至視窗寬或高
  - **指定寬度**：自訂顯示寬度（px）
- **頁面模式**
  - **單頁**：一次顯示一頁
  - **雙頁**：左右雙頁（適合實體漫畫掃圖）
  - **垂直連續 (Webtoon)**：直向捲動，適合條漫／韓漫
- **幻燈片播放**：可設定間隔毫秒數自動翻頁
- **縮放與平移**：滑鼠拖曳平移、滾輪或快捷鍵縮放
- **全螢幕**：F / F11 切換
- **快捷鍵自訂**：在設定中自由更改所有快捷鍵，依個人習慣調整
- **視窗狀態記憶**：關閉應用時自動記住視窗大小與位置

## 支援格式

| 類型 | 副檔名 |
|------|--------|
| 圖片 | `.jpg`、`.jpeg`、`.png`、`.gif`、`.bmp`、`.webp`、`.tiff`、`.tif`、`.avif` |
| 壓縮檔 | `.zip`、`.cbz` |

## 從原始碼執行

### 環境需求

- Node.js（建議 v18+）
- npm

### 快速開始

```bash
# 複製專案後安裝依賴
npm install

# 啟動應用程式
npm start
```

啟動後點選「📁 開啟目錄」或「📦 開啟壓縮檔」選擇漫畫資料來源即可開始閱讀。

## 使用說明

### 開啟漫畫

有三種方式可以開啟漫畫：

1. **工具列按鈕**：點擊「📁 開啟目錄」選擇漫畫所在資料夾，或點擊「📦 開啟壓縮檔」選擇 `.zip` / `.cbz` 檔案
2. **檔案總管**：在檔案總管中雙擊任何支援的圖片格式，即會啟動本應用程式並開啟檔案所在目錄
3. **快捷鍵**：按下 `Ctrl+O` 開啟目錄選擇對話框

### 側邊欄瀏覽

- 左側側邊欄會顯示當前目錄下的**子資料夾**與**圖片檔案**
- 點擊子資料夾可進入瀏覽，點擊「⬆ 上層」返回上層目錄
- 點擊圖片檔案可直接跳到該頁
- 側邊欄邊緣可**拖曳調整寬度**
- 切換目錄時會自動記憶捲動位置，返回上層時還原

### 翻頁瀏覽

- 使用工具列的「◀ 上一頁」、「下一頁 ▶」按鈕
- 或使用鍵盤快捷鍵（方向鍵、PageUp / PageDown）
- 滑鼠滾輪在單頁 / 雙頁模式下可翻頁（滾到邊界時觸發）
- 在 Webtoon 模式下滾輪為正常捲動

### 顯示模式切換

- **適合寬度**：圖片寬度自動填滿閱覽區域
- **適合高度**：圖片高度自動填滿閱覽區域
- **指定寬度**：手動輸入想要的圖片顯示寬度（px）
- 按下 `H` 可快速在「適合寬度」與「適合高度」之間切換

### 頁面模式

- **單頁** (`1`)：一次顯示一張圖片
- **雙頁** (`2`)：同時顯示兩張圖片，適合傳統漫畫左右頁排版
- **垂直連續** (`3`)：所有圖片垂直排列，直向捲動閱讀，適合 Webtoon / 條漫

### 幻燈片播放

1. 設定播放間隔（預設 500 毫秒）
2. 點擊「▶ 幻燈片」或按 `P` 開始自動播放
3. 再按一次停止播放

### 縮放與平移

- **放大**：按 `+` 或 `=`
- **縮小**：按 `-`
- **平移**：按住 `W`/`A`/`S`/`D` 持續移動畫面
- **滑鼠拖曳**：按住滑鼠左鍵拖曳可平移畫面

### 全螢幕

- 按 `F` 或 `F11` 進入 / 離開全螢幕
- 按 `Esc` 可退出全螢幕

### 快捷鍵自訂

1. 點擊工具列上的「⚙ 設定」按鈕
2. 在設定視窗中，點擊想要修改的動作旁的「+」按鈕
3. 按下想要設定的按鍵
4. 點擊已有按鍵旁的「✕」可移除該按鍵
5. 「重設為預設值」可恢復所有預設快捷鍵

## 快捷鍵一覽（預設值）

| 功能 | 按鍵 |
|------|------|
| 開啟目錄 | `Ctrl+O` |
| 上一頁 | `←` / `PageUp` |
| 下一頁 | `→` / `PageDown` / `Space` |
| 第一頁 | `Home` |
| 最後一頁 | `End` |
| 向上移動 | `W` / `↑` |
| 向下移動 | `S` / `↓` |
| 向左移動 | `A` |
| 向右移動 | `D` |
| 單頁模式 | `1` |
| 雙頁模式 | `2` |
| 垂直連續 | `3` |
| 切換適合寬/高 | `H` |
| 放大 | `+` / `=` |
| 縮小 | `-` |
| 幻燈片 | `P` |
| 全螢幕 | `F` / `F11` |

> 所有快捷鍵均可在「⚙ 設定」中自訂修改。
