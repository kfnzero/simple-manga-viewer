# Simple Manga Viewer

簡潔的跨平台漫畫／插畫閱讀器，以 Electron 打造，支援本機資料夾瀏覽與多種顯示模式。

## 功能特色

- **開啟目錄**：選擇本機資料夾，自動掃描並依檔名排序圖片
- **側邊欄瀏覽**：顯示子資料夾與檔案列表，可快速切換章節或單檔
- **顯示模式**
  - **適合寬度 / 適合高度**：自動縮放至視窗寬或高
  - **指定寬度**：自訂顯示寬度（px）
- **頁面模式**
  - **單頁**：一次顯示一頁
  - **雙頁**：左右雙頁（適合實體漫畫掃圖）
  - **垂直連續 (Webtoon)**：直向捲動，適合條漫／韓漫
- **幻燈片播放**：可設定間隔秒數自動翻頁
- **縮放與平移**：滑鼠拖曳平移、滾輪或快捷鍵縮放
- **全螢幕**：F / F11 切換
- **快捷鍵**：翻頁、縮放、切換模式等均可鍵盤操作（詳見應用內「快捷鍵」）

## 支援格式

圖片副檔名：`.jpg`、`.jpeg`、`.png`、`.gif`、`.bmp`、`.webp`、`.tiff`、`.tif`、`.avif`

## 環境需求

- Node.js（建議 v18+）
- npm

## 安裝與執行

```bash
# 複製專案後安裝依賴
npm install

# 啟動應用程式
npm start
```

啟動後點選「開啟目錄」選擇漫畫所在資料夾即可開始閱讀。

## 打包建置

```bash
# 建置 Windows 安裝檔（NSIS）
npm run build:win

# 建置 macOS（僅在 Mac 上執行）
npm run build:mac
```

產出位置：`dist/`（Windows 為 `.exe` 安裝程式，Mac 為 `.dmg`）。

## 快捷鍵摘要

| 功能       | 按鍵 |
|------------|------|
| 開啟目錄   | `Ctrl+O` |
| 上一頁     | `←` / `PageUp` |
| 下一頁     | `→` / `PageDown` |
| 第一頁     | `Home` |
| 最後一頁   | `End` |
| 單頁模式   | `1` |
| 雙頁模式   | `2` |
| 垂直連續   | `3` |
| 切換適合寬/高 | `H` |
| 放大       | `+` / `=` |
| 縮小       | `-` |
| 平移       | `W` / `S` / `A` / `D` 或方向鍵 |
| 幻燈片     | `P` |
| 全螢幕     | `F` / `F11` |
| 快捷鍵說明 | 工具列「快捷鍵」按鈕 |

更多按鍵可在 `renderer.js` 頂部 `keyBindings` 自訂。
