Simple Manga Viewer

點擊「開啟目錄」選擇漫畫資料夾開始閱讀

快捷鍵: Ctrl+O

快捷鍵說明

移動畫面（按住連續移動）

W / S / A / D上 / 下 / 左 / 右 移動
↑ / ↓上 / 下 移動（同 W/S）

翻頁

← / PageUp上一頁 / 上一張
→ / PageDown下一頁 / 下一張
Home第一頁
End最後一頁

頁面模式

1單頁模式
2雙頁模式
3垂直連續模式（Webtoon / 韓漫）

其他

Ctrl+O開啟目錄
H切換適合寬度 / 適合高度
+ / =放大圖像
-縮小圖像
P幻燈片播放 / 停止
F / F11全螢幕切換
Esc關閉對話框 / 退出全螢幕

💡 多組按鍵: 可在 renderer.js 頂部的 keyBindings 物件自由新增任意按鍵組合。
